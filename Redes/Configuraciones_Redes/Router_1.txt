! Configuración básica
enable

line console 0
password cisco
login
exit
line vty 0 4
password cisco
login
exit
service password-encryption
no ip domain-lookup
banner motd #Acceso restringido - Router1#

 no shutdown
 description Conexion a Router2
exit

 version 2
 network 10.0.0.0
 network 192.168.1.0
 no auto-summary
exit

end
