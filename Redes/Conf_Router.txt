*******************************************************
Paso 1: Crear la topología básica
*******************************************************


Agregar dispositivos a la topología:

+ 5 estaciones de trabajo: Agrega cinco PCs al espacio de trabajo.
+ 1 Switch: Agrega un switch para conectar las estaciones de trabajo.
+ 1 Servidor: Agrega un servidor que alojará la página web.
+ Router: Agrega un router para gestionar la red.

Conectar los dispositivos:

+ Conecta cada estación de trabajo al switch utilizando cables de cobre (Cable de consola o de Ethernet).
+ Conecta el servidor al switch utilizando un cable de Ethernet.
+ Conecta el router al switch también con un cable Ethernet.

Configurar las IPs:

+ En el servidor: Asigna una IP estática dentro del rango 192.168.0.0/24, por ejemplo, 192.168.0.2 con máscara de subred 255.255.255.0.

+ En las estaciones de trabajo: Configura la asignación dinámica de IP utilizando el servicio DHCP, que será proporcionado por el router.


*******************************************************
Paso 2: Configurar el router
*******************************************************

Router> enable
Router# configure terminal
Router(config)# interface FastEthernet0/0
Router(config-if)# ip address 192.168.0.1 255.255.255.0
Router(config-if)# no shutdown

Router(config)# ip dhcp pool Politecnico
Router(dhcp-config)# network 192.168.0.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.0.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# exit

Router(config)# ip dhcp excluded-address 192.168.0.1 192.168.0.2

Router(config)# enable secret cisco
Router(config)# line vty 0 4
Router(config-line)# password cisco
Router(config-line)# login
Router(config-line)# exit

        <h2>¿Qué son las redes?</h2>
        <p>Las redes informáticas son un conjunto de dispositivos, como computadoras, impresoras, servidores y otros, interconectados entre sí mediante cables, Wi-Fi o cualquier otro medio, con el fin de compartir información, recursos y servicios. Estas conexiones permiten que los dispositivos se comuniquen y colaboren entre sí de manera eficiente.</p>
    </section>

    <section id="para-que-sirven">
        <h2>¿Para qué sirven las redes?</h2>
        <p>Las redes son esenciales para facilitar la comunicación entre dispositivos. Gracias a ellas, se pueden compartir archivos, acceder a internet, utilizar impresoras compartidas, y utilizar servicios de almacenamiento y aplicaciones en la nube. Además, permiten la integración de sistemas a gran escala, como en empresas y organizaciones.</p>
    </section>

    <section id="que-es-dhcp">
        <h2>¿Qué es el protocolo DHCP?</h2>
        <p>El protocolo DHCP (Dynamic Host Configuration Protocol) es un protocolo de red utilizado para asignar automáticamente direcciones IP a los dispositivos que se conectan a una red. El servidor DHCP asigna una IP dinámica dentro de un rango predefinido, lo que facilita la configuración de las redes, ya que los dispositivos no necesitan configurarse manualmente con una IP estática.</p>
        <p>Este protocolo también puede proporcionar otros parámetros de configuración, como la puerta de enlace predeterminada (gateway) y los servidores DNS. DHCP es utilizado en la mayoría de las redes modernas debido a su simplicidad y eficiencia.</p>
